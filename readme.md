# sudo-bot

general purpose discord bot that executes commands and returns its output


### install

`npm install`

`pm2 --name sudo-bot start ./index.mjs`

`pm2 save`


### uninstall

`pm2 delete sudo-bot`

`pm2 save`


### invite

`https://discord.com/api/oauth2/authorize?client_id=1156246056019968020&permissions=0&scope=bot`
